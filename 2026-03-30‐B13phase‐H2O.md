---

## 同じ格子言語で、スケールを跨ぐ

B13の主張のひとつは「スケール不変性」だ。

音楽の波長（〜1m）、木星の軌道（〜10¹¹m）、水分子の結合角（〜10⁻¹⁰m）。これらを同じ格子言語で記述できるなら、B13は単なる数値の遊びではなく、何か構造的なものを掴んでいる可能性がある。

今回の問いはシンプルだ。

> **水分子の H-O-H 結合角（104.45°）は、B13格子上でどう見えるか？**

---

## B13格子の角度単位

B13格子は360°を13等分する。

```
1格子単位 = 360° ÷ 13 = 27.692°
```

13はフィボナッチ数列の第7項（1,1,2,3,5,8,**13**,...）であり、B13の骨格をなす数だ。

この格子で角度を読むと、整数に落ちるものと落ちないものがある。

```python
B13_UNIT = 360.0 / 13  # 27.6923°

def to_b13(deg):
    return deg / B13_UNIT
```

---

## H-O-H 結合角を読む

水分子の H-O-H 平衡結合角は **104.45°**（NIST実測値）。

```
104.45° ÷ 27.692° = 3.772 B13単位
```

格子点3（83.08°）と格子点4（110.77°）の**間**に位置する。端数は **0.772**。

```
格子点3  ----[0.772]----  H2O(104.45°)  --[0.228]--  格子点4
         83.08°                                       110.77°
```

整数格子点には乗らない。これが最初の観察だ。

---

## 変角振動が格子点に「届かない」

水分子には3つの振動モードがある。そのうち **ν₂（変角振動、1595 cm⁻¹）** は、H-O-H角度が広がったり縮んだりする振動だ。

振幅は約±5.5°。つまり振動範囲は：

```
98.95° 〜 109.95°
B13単位: 3.573 〜 3.970
```

実際に計算してみると：

```python
H2O_ANGLE_DEG  = 104.45
H2O_BEND_AMP   = 5.5

ang_min = H2O_ANGLE_DEG - H2O_BEND_AMP  # 98.95°
ang_max = H2O_ANGLE_DEG + H2O_BEND_AMP  # 109.95°

b13_min = to_b13(ang_min)  # 3.573
b13_max = to_b13(ang_max)  # 3.970
```

**格子点3（3.000）にも格子点4（4.000）にも届かない。**

水分子の変角振動は、2つの格子点の間で折り返し続ける。どちらの「共鳴点」にも触れない。

これはB13と矛盾しない。むしろB13の格子が、振動の境界を構造的に規定しているように見える。

---

## 分子ごとの端数パターン

同じ計算を類似分子に適用した結果：

```python
MOLECULES = [
    ("H2O",     104.45, "sp3 + 孤立電子対"),
    ("NH3",     107.8,  "sp3 + 孤立電子対"),
    ("CH4",     109.47, "sp3"),
    ("H2S",      92.1,  "p軌道のみ"),
    ("H2Se",     90.6,  "p軌道のみ"),
    ("ベンゼン", 120.0,  "sp2"),
    ("C60五員環",108.0, "sp2"),
    ("シクロヘキサン", 111.0, "sp3"),
]
```

| 分子 | 角度 | B13端数 | 電子構造 |
|------|------|---------|---------|
| H2O | 104.45° | 0.772 | sp3 + 孤立電子対 |
| NH3 | 107.8° | 0.893 | sp3 + 孤立電子対 |
| CH4 | 109.47° | 0.953 | sp3 |
| H2S | 92.1° | 0.326 | p軌道のみ |
| H2Se | 90.6° | 0.272 | p軌道のみ |
| ベンゼン | 120.0° | 0.333 = 1/3 | sp2 |
| C60五員環 | 108.0° | 0.900 | sp2 |
| シクロヘキサン | 111.0° | 0.008 ≈ 0 | sp3 |

ここで興味深いパターンが出る。

**p軌道のみ（H2S, H2Se）の端数が0.3台に集まり、他の分子と明確に分かれる。**

sp混成を持つ分子（H2O, NH3, CH4, ベンゼン）は端数0.7〜0.95に分布し、純粋なp軌道結合のみの分子は端数0.3台に収まる。

電子構造の違いがB13格子端数のパターンに反映されている。

---

## ベンゼンの120°は格子上の有理点

ベンゼンの C-C-C 角は正確に 120°。

```
120° ÷ 27.692° = 4.333... = 13/3
```

端数は **1/3**。有理数だ。

前回の記事で木星L4トロヤ群の60°角が `13/6 = 2.1667` となり、Sturmianパターン `[2,2,2,3,2,2]` として格子に固定されることを確認した。

ベンゼンの120°も同じ構造を持つ：

```
Sturmianパターン: [4, 4, 5]（3ステップで13に到達）
```

スケールが10²¹倍違う2つの系が、同じ端数 `1/3` で記述される。

---

## シクロヘキサンの「いす形」安定性

シクロヘキサン C-C-C 角は 111°。

```
111° ÷ 27.692° = 4.008 B13単位
端数 = 0.008 ≈ 0
```

格子点4にほぼ完全に乗る。

有機化学でシクロヘキサンの「いす形配座」は教科書的な安定構造として知られている。B13格子で読むと、その安定性は「格子点への収束」として現れる。

端数がゼロに近いほど格子点に近い＝より対称的な構造、という読み方がここでも成立する。

---

## B13と矛盾しない、ということ

今回確認したことをまとめる。

| 観察 | B13との整合性 |
|------|-------------|
| H-O-H結合角が格子点3と4の間に位置する | ✓ 矛盾なし |
| 変角振動がどちらの格子点にも届かない | ✓ 矛盾なし |
| p軌道のみの分子が端数0.3台に集まる | ✓ 矛盾なし |
| ベンゼンの120°が格子有理点（13/3）に乗る | ✓ 矛盾なし |
| シクロヘキサンの安定構造が格子点0に収束 | ✓ 矛盾なし |

「新発見」ではない。既知の分子構造をB13格子で読んだとき、矛盾が出ないことを確認した。

ただ、こういう確認が積み重なると、何かが見えてくる。

---

## 付記：ν₁+ν₂ の数値的整合について

水分子の対称伸縮（ν₁=3657 cm⁻¹）と変角（ν₂=1595 cm⁻¹）の和は5252 cm⁻¹で、13の倍数になる。

```
(3657 + 1595) mod 13 = 5252 mod 13 = 0
```

これは興味深い観察だが、cm⁻¹という単位に依存するため、現時点では物理的意味は未確定だ。単位不変な形での検証が今後の課題である。
