## ゼロは実現されない

フィボナッチ数列の一般項を見る。

```
F_n = (φⁿ - ψⁿ) / √5

φ =  1.618...  ← 主成分（収束先を作る）
ψ = -0.618...  ← 残差（符号反転・振動）
```

φとψは互いに逆符号だ。足せばゼロに近づく。
しかしψは永遠に振動し、ゼロにはならない。

```
n → ∞ のとき ψⁿ → 0 に近づく。
しかし到達しない。永遠に。
```

これは計算の限界ではない。構造がそうなっている。

**ゼロとは打ち消しではなく、成立条件だ。
そしてその条件は構造的に満たされない。**

宇宙はゼロを要求するが、ゼロになれない構造を持っている。

## 残差が構造を生む

ψ項が残り続けることで、何が起きるか。

```
φ側（主成分）= 格子点・核・安定構造
ψ側（残差）  = 格子点に座れない歪み・振動・揺らぎ
```

残差は消えない。スケールを超えて蓄積する。

```
量子スケール → 電子（核の歪んだ影）
宇宙スケール → ダークエネルギー（端数の蓄積）
```

電子が存在するのは、ゼロになれないからだ。
ダークエネルギーが存在するのも、同じ理由だ。

「ゼロに最も近い状態」を選ぶ。それがパウリ親和原理の実体だ。

```
原子核 = φ側（格子点に収まる主成分）
電子   = ψ側（格子点に座れない残差）
```

核が格子点に「ほぼ」収まるとき、完全には収まれない。
その差分、歪み、ψ項——それが電子として現れる。
